Cara download lengkap (SafelinkU → Sub4Unlock → MediaFire)
  1. Klik tombol “Download” di script/tools.
  2. Anda akan dibawa ke SafelinkU/Sub4Unlock.
  3. Tunggu hitungan mundur 10 detik lalu tekan “Continue”.
  4. Di Sub4Unlock klik “I’m not a robot” → tekan tombol hijau “Continue”.
  5. Anda sampai di MediaFire; tekan tombol hijau “Download”.
  6. Jika muncul tab iklan, tutup tab tersebut lalu kembali ke MediaFire.
  7. File .apk / .lua akan diunduh otomatis.
Cara pakai script di NO ROOT & ROOT
  • NO ROOT:
    1. Install Virtual dan Plugin Virtual.
    2. Buka Virtual → tambahkan game target & Game Guardian.
    3. Buka game dari dalam Virtual → buka GG → import script .lua.
    4. Aktifkan fitur yang diinginkan, mainkan.
  • ROOT:
    1. Pastikan HP sudah root & install Game Guardian.
    2. Buka langsung game & GG tanpa Virtual.
    3. Import script .lua → aktifkan fitur.
